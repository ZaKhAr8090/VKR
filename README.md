# Метод генерации уточненного датасета с применением визуальных языковых моделей в решении задач сегментации объектов в условиях наличия аномалий

Данный репозиторий содержит реализацию комбинированного подхода к подготовке наборов данных для задач компьютерного зрения. Исследование решает две взаимосвязанные проблемы: автоматическое обнаружение и удаление аномалий из обучающих выборок, а также генерация высококачественной разметки сегментации на основе более доступной разметки детекции.

Подход особенно актуален для прикладных задач, где качественная пиксельная разметка либо отсутствует, либо её создание требует значительных временных затрат, а исходные датасеты могут содержать различные искажения, снижающие обобщающую способность моделей.

---

## Основные компоненты

### 1. Поиск и очистка от аномалий (ПМ_А)

Разработанный метод не требует предварительного обучения на чистых данных, не чувствителен к размерности и обеспечивает стабильную работу в различных условиях. Алгоритм основан на следующих шагах:

1. **Векторизация изображений** — извлечение признаковых векторов с помощью визуально-языковых моделей
2. **Снижение размерности** — применение PCA для стабилизации последующей кластеризации
3. **Кластеризация** — выделение групп визуально схожих изображений алгоритмом k-means (число кластеров определяется силуэтным анализом)
4. **Оценка аномальности** — для каждого изображения вычисляется косинусное сходство с медианными векторами кластеров и глобальным медианным вектором. Изображения сортируются по возрастанию максимального сходства — аномалии оказываются в начале списка

В текущей работе он сравнивался с такими методами, как автокодировщики, Isolation Forest, One-class SVM, CLIP.

### 2. Переход от детекции к сегментации (ПМ_П)

Предложенный метод автоматически преобразует разметку в формате ограничивающих рамок (bounding boxes) в полноценные маски сегментации:

1. Готовая разметка детекции (bbox) подаётся в фундаментальную модель SAM3 в качестве визуальных промптов
2. SAM3 генерирует точные маски сегментации внутри каждой рамки
3. Полученные маски используются для обучения лёгкой модели YOLO11m-seg, которая затем заменяет ресурсоёмкую связку на этапе инференса

## Датасеты

| Датасет | Описание |
|---------|----------|
| **WaRP** | Промышленные отходы на конвейерной линии (высокий уровень визуального сходства, множественные перекрытия, сильный дисбаланс классов) |
| **MVTec AD** | Промышленный датасет для оценки методов поиска аномалий (15 категорий, локальные дефекты) |

## Результаты экспериментов

### Поиск аномалий 

Поиск глобальных аномалий на датасете WaRP: ПМ_А превосходит все сравниваемые алгоритмы в двух из трех тестов, в том числе решения, обучавшиеся на чистых данных                    
Поиск локальных аномалий на датасете MVTec AD: ПМ_А продемонстрировал результаты, сопоставимые с рассматриваемыми решениями, с незначительным отставанием в отдельных случаях

### Переход от детекции к сегментации (датасет WaRP)

| Метрика | YOLO11m (детекция) | YOLO11m-seg (после обучения на масках SAM3) | Δ |
|---------|-------------------|----------------------------------------------|---|
| **mAP50** (по рамкам) | 0.524 | 0.544 | **+2.0%** |
| **mAP50-95** (по рамкам) | 0.405 | 0.460 | **+5.5%** |

---

Обработка одного изображения (на GPU T4 X2) связкой YOLO11m + SAM3 занимает 76,16 мс (15,4 мс — детекция YOLO11m, 60,76 мс — сегментация SAM3). Модель YOLO11m‑seg, обученная на масках SAM3, обрабатывает одно изображение за 20,6 мс. В пересчете на FPS это соответствует 13,13 и 48,54 соответственно, что означает более чем трехкратное превосходство по скорости.
