# Генерация уточнённого датасета с применением VLM для сегментации и очистка от аномалий


Данный репозиторий содержит реализацию исследовательского пайплайна по генерации разметки сегментации с использованием связки детектор‑сегментатор (YOLO12s + SAM3) и последующей дообучки модели сегментации YOLO11s‑seg на полуавтоматической разметке, а также подхода к выявлению аномалий и отравления данных с помощью VLM‑эмбеддингов и аномалий‑детекторов.

## Основные идеи

- **Генерация масок сегментации**  
  Связка моделей `YOLO12s` (детекция) и `SAM3` (сегментация по bbox). YOLO12s детектирует объекты, полученные рамки передаются в SAM3, который внутри них строит точные маски.

- **Обучение одноэтапной модели сегментации**  
  Сгенерированные маски SAM3 используются как разметка для обучения `YOLO11s-seg`. Это позволяет заменить медленный двухэтапный пайплайн на быстрый одноэтапный ценой потенциального переноса ошибок SAM3.

- **Очистка данных от аномалий и отравлений**  
  Для повышения качества обучения применяются три метода поиска искажённых изображений:
  - `IsolationForest`
  - `OneClassSVM` (с PCA и без)
  - **Предложенный многоступенчатый метод** (PCA + k‑means + косинусное сходство с медианными векторами кластеров)

## Экспериментальные датасеты

| Датасет | Описание | Классов | Изображений | Разметок детекции |
|---------|----------|---------|--------------|-------------------|
| **Color‑Cap** | Цветные крышечки на конвейере (высокая контрастность, мало шума) | 9 | 2187 | 46467 |
| **WaRP** | Промышленные отходы (перекрытия, визуальное сходство) | 28 | 2974 | 10374 |
| **MVTec AD** | Промышленный датасет для поиска аномалий| - | - | - |

## Используемые VLM

- **DINOv2 (base)** — визуальный трансформер, выход CLS-токена размерностью 768.
- **CLIP (ViT-B-32)** — мультимодальная модель, вектор изображения размерностью 512.

## Результаты поиска аномалий

### Датасет WaRP (5 аномалий на 160 кропов, показано положение аномалий в ранжированном списке)

| Метод | VLM | Позиции аномалий |
|-------|-----|------------------|
| **Предложенный метод** | DINOv2 | 1 3 4 10 |
| | CLIP | 1 4 6 |
| IsolatedForest | DINOv2 | 4 6 |
| | CLIP | 5 7 |
| OneClassSVM (PCA=25) | DINOv2 | - (-) |
| | CLIP | 5 (-) |

> *OneClassSVM без PCA не показал ни одной аномалии.*

### Датасет Color‑Cap (3 аномалии на 40 чистых изображений)

| Метод | VLM | Позиции аномалий |
|-------|-----|------------------|
| **Предложенный метод** | DINOv2 | 1 3 4 |
| | CLIP | — |
| IsolatedForest | DINOv2 | 1 2 |
| | CLIP | 1 2 |
| OneClassSVM (PCA=25) | DINOv2 | 1 (3) |
| | CLIP | - (1 3) |


Числа в ячейках указывают на позицию изображения в ранжированном списке по вероятности быть аномальным — чем меньше значение, тем выше вероятность, что модель определила изображение как аномальное (т.е. значение 1 означает наивысшую уверенность в аномальности)

Предложенный многоступенчатый метод оказался не столько эффективен в поиске отравленных данных

Финальное тестирование моделей Yolo-seg показывает, что несмотря на возможные сложности с датасетами, связка YOLO12s + SAM3 показывает себя эффективно в задачах перехода от детекции к сегментации
